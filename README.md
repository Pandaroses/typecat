# Typecat

typecat is a terminal typing test made by me as my first rust project

## Installation

Use the aur to install typecat.

```bash
 yay -S typecat
```

or manual installation

```bash
  git clone https://github.com/pandaroses/typecat
  cd typecat
  makepkg -si
```


to run

```bash
 typecat
```
